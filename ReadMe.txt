# Morty

A cross platform game engine based on Vulkan. Only for personal study and research.

## Getting Started

### Dependencies

- CMake

- Ninja

### Installing

```
    git clone https://github.com/FrostyLeaves/Morty.git

    cd ./Morty/Morty

    cmake --preset [Windows | Linux | Macos-Xcode]

```
